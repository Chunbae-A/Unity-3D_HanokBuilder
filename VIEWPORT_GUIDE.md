## 화면 구성

```
┌──────────┬──────────────────────────┬──────────┐
│  왼쪽    │       가운데 뷰포트      │  오른쪽  │
│  에셋    │  [3D][위↓][앞][뒤][우]  │  인스펙터│
│  패널    │                          │  패널    │
│          │   ← 편집 공간 (마당) →  │          │
│  썸네일  │                          │  Transform│
│  그리드  │  [선택][이동][회전][삭제]│  정보    │
└──────────┴──────────────────────────┴──────────┘
```

| 영역 | 기능 |
|------|------|
| 왼쪽 패널 | 한옥 부재 에셋 라이브러리. 카테고리 탭 + 썸네일 그리드 |
| 가운데 뷰포트 | 3D 편집 공간. 카메라 조작·오브젝트 배치·조작 |
| 오른쪽 패널 | 선택 오브젝트의 이름·위치·회전·크기 편집 |

## 카메라 조작

### 마우스

| 동작 | 기능 |
|------|------|
| **우클릭 + 드래그** | 3D 오비트 (카메라 회전) |
| **Alt + 좌클릭 + 드래그** | 3D 오비트 (Blender / Unity 스타일) |
| **중간버튼 + 드래그** | 패닝 (카메라 수평 이동) |
| **Shift + 우클릭 + 드래그** | 패닝 (마우스 중간버튼 없을 때 대안) |
| **스크롤 휠** | 줌인/줌아웃 (커서 방향으로 이동) |
| **Ctrl + 스크롤** | 정밀 줌 (세밀한 근접 작업용) |

> **줌 → 커서 동작**: 스크롤로 줌인하면 카메라가 마우스 커서가 가리키는 지점으로  
> 이동합니다. 패닝 없이 작업 영역에 바로 근접할 수 있습니다.

### 키보드 이동

| 키 | 기능 |
|----|------|
| **W / S** | 앞뒤 이동 |
| **A / D** | 좌우 이동 |
| **E / Q** | 위아래 이동 |
| **Shift + 이동키** | 3× 빠른 이동 |

### 오비트 중심 자동 조정

오브젝트를 **클릭해서 선택하면** 카메라 오비트 중심(피벗)이 해당 오브젝트  
방향으로 부드럽게 이동합니다. 선택한 오브젝트를 중심으로 자연스럽게  
카메라를 돌릴 수 있습니다.

## 툴 모드

뷰포트 좌측 하단 툴바에서 클릭하거나 단축키로 전환합니다.

| 툴 | 단축키 | 기능 |
|----|--------|------|
| **선택** | `1` | 오브젝트 클릭 선택. 선택 후 드래그로 즉시 이동 |
| **이동** | `2` | 뷰포트 드래그로 오브젝트 이동 |
| **회전** | `3` | 3축 링 기즈모 표시. 링 드래그로 회전 |
| **삭제** | `4` | 선택 오브젝트 삭제 |

> **선택 모드 팁**: 오브젝트를 클릭 후 바로 드래그하면 이동 툴로 전환하지  
> 않아도 이동할 수 있습니다 (5px 이상 드래그 시 자동 이동 활성화).

## 에셋 배치 워크플로

```
1. 왼쪽 패널에서 에셋 클릭
       ↓
2. 에셋이 마당 중앙에 배치됨 (y=0)
   + 카메라가 해당 에셋으로 자동 포커스
       ↓
3. 선택/이동 툴로 위치 조정
   or 우측 패널 Transform 입력창에 수치 직접 입력
       ↓
4. 회전 툴(3)로 방향 조정
   — X(빨강) 링 : 앞뒤 기울기
   — Y(초록) 링 : 좌우 회전 (가장 많이 사용)
   — Z(파랑) 링 : 롤 방향
       ↓
5. F키로 선택 오브젝트 클로즈업
   Z키로 전체 배치 상태 한 번에 확인
```

## 뷰 전환

### 뷰포트 우상단 버튼 또는 Numpad 단축키

| 버튼 | Numpad | 뷰 |
|------|--------|----|
| **3D** | `Numpad 0` / `Numpad 5` | 원근(Perspective) — 기본 편집 뷰 |
| **위↓** | `Numpad 7` | 탑뷰(직교) — 평면 배치 확인 |
| **앞** | `Numpad 1` | 정면(직교) — 높이 정렬 확인 |
| **뒤** | `Numpad 9` | 후면(직교) |
| **우** | `Numpad 3` | 우측면(직교) — 깊이 정렬 확인 |

> **직교 뷰에서 우클릭** 하면 자동으로 원근 뷰로 전환됩니다.

### 활용 예시

- 기둥 간격 확인 → **위↓(Numpad 7)** 탑뷰
- 지붕 높이 맞추기 → **앞(Numpad 1)** 정면뷰
- 처마 깊이 조정 → **우(Numpad 3)** 우측뷰
- 전체 구성 확인 → **3D(Numpad 0)** 원근뷰

## 회전 기즈모

회전 툴(`3`) 선택 시 오브젝트 주변에 3색 링이 표시됩니다.

```
  🔴 X축 (빨강) — 피치(앞뒤 기울기)   위아래 드래그
  🟢 Y축 (초록) — 요(좌우 회전)       좌우 드래그    ← 가장 많이 사용
  🔵 Z축 (파랑) — 롤(좌우 기울기)     좌우 드래그
```

- 링에 **마우스 호버** 시 흰색으로 강조 표시
- 링을 **클릭 & 드래그** 해서 회전
- 링 크기는 **에셋 크기에 자동 비례** (작은 부재 = 작은 링)

### FBX cm-스케일 이슈

Unity에 임포트된 FBX가 cm 단위로 모델링된 경우 bounds가 100× 크게 잡힐 수 있습니다.  
이 경우 기즈모가 자동으로 **카메라 거리 기반 폴백**으로 전환해 적절한 링 크기를 유지합니다.

> **해결책**: FBX 임포트 설정에서 `Scale Factor = 0.01` 로 설정 (#10 참조)

## 배치 팁

### 정밀한 수치 입력
오른쪽 패널의 **Position / Rotation / Scale 입력창**에 직접 수치를 입력하면  
드래그보다 정확하게 오브젝트를 배치할 수 있습니다.

### 줌 → 커서 활용
부재 하나를 자세히 보려면 해당 부재 위에 마우스를 올리고 스크롤하세요.  
카메라가 커서 위치 방향으로 줌인됩니다.

### 복잡한 구성 편집 순서 (권장)
```
① Z키로 전체 보기 → 배치 현황 파악
② 수정할 오브젝트 클릭 → 자동 포커스
③ Numpad 1/3/7로 정면/측면/탑뷰 전환 → 정밀 조정
④ F키로 클로즈업 → 세부 회전/이동
⑤ Z키로 다시 전체 보기 → 균형 확인
```

### 한국 전통 배치 참고
- **기둥** 배치 후 → **보/도리** 순으로 상부 구조
- **기단** 위에 오브젝트를 올릴 때 Y위치를 0으로 맞추면 기단 위에 정렬됨
- 탑뷰(Numpad 7)에서 좌우 대칭을 확인하며 배치
