Local naming policy
- Runtime-loadable CultureMetaverse object FBX files use ASCII-safe Unity names like SM_CM_Object_001.
- Original Korean titles and source URLs are preserved in culture_object_board_sources.tsv and object_direct_fbx_sources.tsv.
- culture_metaverse_index.txt supplies the display name shown in the module library, so local filenames can stay safe without losing the source names.

Package handling
- KoreanTraditionalFestival Unity package is represented by expanded prefabs under Props/KoreanTraditionalFestival/Prefabs.
- Merchant/environment FBX files are represented under Props/MerchantAndEnvironment.
- Interactive Unity/Unreal engine packages are recorded in culture_object_board_sources.tsv but are not expanded into module-library FBX entries.
